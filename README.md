# Nomad Fest Workshop — Vibe Coding Live Demo

**Event:** Nomad Fest
**Format:** 20-minute live demo workshop
**Audience:** Non-technical entrepreneurs and digital nomads
**Goal:** Leave audience feeling "I could do that" + aware of Code Explorer on WHOP

---

## What We're Building

One project, two phases:

**Phase 1 — Speed Demo:** Build a pickleball website from scratch using a single prompt. Entire site in under 2 minutes. This is the wow moment.

**Phase 2 — Plan Mode:** Add a reaction game to the same site using Plan Mode first, to show how to think before you build.

## Quick Reference — Prompts

**Prompt 1 (Basic structure — intentionally plain):**
> Build a simple pickleball website with games, leaderboard, and a join form. Keep it basic for now.

**Prompt 2 (Plan Mode — full brand transformation + game):**
